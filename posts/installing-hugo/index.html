Installing Hugo

Thu, May 31, 2018

This is a quick-start guide to setting up Hugo with GitHub pages, since it took too much effort to find the full instructions anywhere else.

  1. Install Hugo from the releases page.
  2. Run hugo new site blog to make the skeleton of your blog website.
  3. Install a theme you want. I just did cd themes and git clone https://github.com/spf13/hyde. More info about themes here.
  4. Update the config.toml generated from step 2 to use the theme. I.e, add theme = "hyde".
  5. Make a new post using hugo new posts/installing-hugo.md. Edit its content to be whatever you want.
  6. Run hugo server -D and browse to http://localhost:1313/ to make sure the theme and content are as you expected. You can peruse the other files to edit the description of the website or refer to this website’s source code to see additional options.
